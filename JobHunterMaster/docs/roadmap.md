# Roadmap de Entrega

## Fase 1 (completada en este MVP)
- stack Python definido
- estructura de solución
- contratos JSON base
- perfiles básicos
- repositorio de documentos con hash + PDF derivado
- API local mínima para UiPath
- dashboard operativo básico (ES/EN + light/dark)
- export de backup en bundle zip

## Fase 2 (siguiente iteración)
- formularios completos de experiencia, educación, skills, idiomas
- generador CV con plantillas ATS/Harvard/IT/Moderno
- import de backup validado y versionado
- cola de tareas para búsquedas automáticas

## Fase 3 (integración avanzada UiPath)
- catálogo de alias por portal
- manejo fino de captcha y reanudación
- scoring híbrido reglas + IA
- métricas y auditoría
